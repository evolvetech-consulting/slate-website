# Slate — marketing site

Landing page for **Slate**, the invoice-to-AccuLynx automation for contractors.

- Single self-contained `index.html` (inline CSS + JS, no build step, no dependencies).
- Light/dark theme, animated "how it works" demo, responsive.
- All names, companies, addresses and figures shown are fictional and for illustration only.

## Deploy

Served as a static site via GitHub Pages from the `main` branch root.
Live at: https://evolvetech-consulting.github.io/slate-website/

To preview locally, just open `index.html` in a browser.
